路由守卫调试信息

当访问 /base/vip 时，请在浏览器控制台查看以下信息：

  1. 检查后台token是否存在： document.cookie.includes('Admin-Token')
  2. 检查AI token是否存在： localStorage.getItem('ai_token')
  3. 检查AI登录状态： localStorage.getItem('ai_isLoggedIn')
  4. 检查路径匹配： '/base/vip'.startsWith('/ai/') (应该是 false)

预期行为：

  • ✅ 路径不以 /ai/ 开头 → 不走AI逻辑
  • ✅ 没有后台token → 重定向到 /login
  • ❌ 如果跳转到 /ai/login → 说明有其他问题

调试命令：

// 在浏览器控制台执行
console.log('后台token:', document.cookie.includes('Admin-Token'))
console.log('AI token:', localStorage.getItem('ai_token'))
console.log('AI登录状态:', localStorage.getItem('ai_isLoggedIn'))
console.log('路径检查:', '/base/vip'.startsWith('/ai/'))
